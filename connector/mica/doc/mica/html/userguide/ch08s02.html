SimpleAgentFrame

SimpleAgentFrame is a extension of JFrame that can be used to provide an agent with a simple GUI.

The main benifits of the SimpleAgentFrame is that is can configure its size, position and title according to parameters given to the agent and that when the window is close it will call terminate on the agent.
